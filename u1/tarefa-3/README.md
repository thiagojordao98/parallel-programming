# Tarefa 3: Cálculo e Aproximação de π

## Descrição do Projeto

Este repositório contém a implementação da **Tarefa 3**, cujo objetivo é desenvolver um programa em C para calcular uma aproximação de π (Pi) utilizando séries matemáticas. A partir dessa implementação, será feita uma análise quantitativa e qualitativa do desempenho e da precisão do algoritmo.

## Objetivos da Atividade

- **Implementação Computacional:** Desenvolver um ou mais programas em C (ex: `pi.c` e `pi_machin.c`) que aproximem o valor de π usando séries matemáticas definidas.
- **Medição e Variação:** Executar os algoritmos variando o número de iterações e registrar os respectivos tempos de execução.
- **Análise de Acurácia:** Comparar os resultados obtidos com o valor real de π, avaliando como o ganho de precisão está diretamente atrelado ao aumento do esforço computacional (mais iterações).
- **Reflexão sobre Aplicações Reais:** Preparar o terreno para um relatório reflexivo (`analise.md`) sobre como esse trade-off entre tempo de processamento e precisão se repete em aplicações reais complexas, tais como **simulações físicas** e algoritmos de **Inteligência Artificial (IA)**.

## Estrutura Atual

- `pi.c` / `pi_machin.c`: Códigos-fonte em C com as implementações das aproximações matemáticas.
- `analise.md`: Arquivo base que reunirá as métricas coletadas e a reflexão final da tarefa.
